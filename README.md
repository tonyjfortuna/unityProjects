# 🎮 unityProjects

Unity projects built from Unity Learn's guided courses, where the course provides a project goal and starter assets, but the gameplay code and design decisions are left to the developer.

## 🗂️ Contents

- **SpriteFlight**: A 2D arcade flight game with physics based movement, a scoring system, and a persistent high score. [Play it](https://play.unity.com/en/games/28d8db36-f336-4172-9c65-0c1b41410e08/sprite-flight)
- **GetStartedWithUnity**: A 3D collectible game with a chase mechanic, dynamic pickup motion, and movement based audio. [Play it](https://play.unity.com/en/games/bd218536-3f18-4f32-adaa-34920fc09af5/get-started-with-unity)
- **RollABall**: The classic Roll a Ball tutorial project, extended with an enemy that actively hunts the player using NavMesh pathfinding, not part of the base tutorial. Not published, local project only.
